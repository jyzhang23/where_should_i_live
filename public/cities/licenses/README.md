# Image Licenses

Store license information for each city image here.

## Template

For each image, create a file named `{city-id}.txt` with the following information:

```
Image: {city-id}.jpg
Source: {URL where image was obtained}
License: {License type, e.g., Unsplash License, CC0, CC BY 4.0, etc.}
Author: {Photographer name}
Attribution: {Required attribution text, if any}
Date Obtained: {YYYY-MM-DD}
```

## Example: new-york-city.txt

```
Image: new-york-city.jpg
Source: https://unsplash.com/photos/xxxxx
License: Unsplash License
Author: John Doe
Attribution: Photo by John Doe on Unsplash
Date Obtained: 2026-01-29
```

## License Types

### Unsplash License
- Free to use for commercial and non-commercial purposes
- No attribution required (but appreciated)
- Cannot be sold without modification

### Pexels License  
- Free to use for any purpose
- No attribution required
- Cannot be sold unaltered

### CC0 (Public Domain)
- No restrictions
- No attribution required

### CC BY 4.0
- Free to use with attribution
- Must credit the creator
